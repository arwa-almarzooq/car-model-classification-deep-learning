This project is a deep learning computer vision system that identifies the car model directly from an image.

- Stanford Cars Dataset
- Images organized by car model and class
- Train / Test split

## Approach
- Image classification using deep learning
- Dataset loading and preprocessing
- Training and evaluation using PyTorch

## Features
- Recognizes car models from raw images
- Handles large multi-class classification
- Trained on real-world car images
